## 13. Pi Review Loop 按审查检查点增量展示 Agent 改动

## 17. PWA Sense Bridge 将手机动作转换为 LLM 结构化事件

PWA Sense Bridge 在本地把吹气、转动、摇晃和长按处理成带强度、方向与时间戳的语义事件，再由宿主选择是否注入任意 LLM 对话后端。它不上传录音，采用姿态校准、阈值与冷却，并要求权限由用户手势触发，同时将传感器事件作为低优先级结构化输入，避免伪装成用户说出的文字。

来源：https://github.com/0xblewalker/pwa-sense-bridge

## 18. RMUC OfflineRL 用四百万条赛场记录训练战术决策

RMUC OfflineRL 基于 613 局 RoboMaster 区域赛、约 400 万条逐秒状态记录，构建从日志解析、161 维观测和 10 维战术动作到多卡训练、离线评估与部署的完整框架。项目比较 IQL、行为克隆和 Decision Transformer，发现模型能以 74.6% 准确率选择交战目标，但也明确报告 IQL 与 Decision Transformer 相比行为克隆尚无可测增益。

来源：https://github.com/Harkerbest/RMUC-OfflineRL

## 19. Fable Thinking 将推理纪律封装为可执行插件

Fable Thinking 把快速前置检查、多假设诊断、具体过程模拟、对抗式自审、硬约束逐项验证和置信度校准整理成 Claude Code Skill。它不声称提升模型能力，而是用机械流程减少流畅但未经证据支持的回答，并可将相同协议迁移到 Codex、Gemini 和本地模型等运行时。

来源：https://github.com/mrgoonie/fable-thinking
