## 7. MIT 与 Microsoft 研究 Murakkab，优化 AI agent 工作流成本

MIT 报道称，Murakkab 可以自动为多步骤 agent 工作流选择模型、工具、硬件和资源配置，从而降低计算、能耗和成本。随着 agent 工作流变复杂，模型调度、工具选择和资源编排会成为重要基础设施。

来源：https://www.eecs.mit.edu/improving-the-speed-and-energy-efficiency-of-ai-agents/

## 8. MIT 讨论 agentic AI 的定义、风险与技术瓶颈

MIT 采访指出，agentic AI 的核心是“能在世界中采取行动”，但当前 agent 仍面临训练数据不足、任务验证不足、隐私泄露和技能退化风险。短期内，最成熟的落地场景仍然是 coding agent，因为任务边界和反馈信号更清晰。

来源：https://www.eecs.mit.edu/qa-what-is-agentic-ai-today-and-what-do-we-want-it-to-be/
