## 18. arXiv：Online Safety Monitoring for LLMs 研究部署时在线安全监控

这篇论文研究如何在 LLM 部署时做实时安全监控：把外部 verifier 的信号转化为告警决策，并通过风险控制校准阈值。作者在数学推理和红队数据集上测试，发现简单阈值监控就能接近更复杂的顺序假设检验方法。这个方向适合落地到生产系统的安全防线。

来源：https://arxiv.org/abs/2607.02510

## 19. arXiv：ReContext 改进长上下文推理中的证据利用

ReContext 提出一种免训练推理方法，用模型内部相关性信号构建 query-conditioned evidence pool，并在最终生成前重新播放关键证据。它试图解决“模型有长上下文窗口但没有有效使用上下文”的问题。论文在 128K 上下文长度的多个数据集上测试，重点是提高证据利用率。

来源：https://arxiv.org/abs/2607.02509

## 20. arXiv：Steerability via constraints 研究 coding agent 的可约束监督

这篇论文讨论如何通过约束提升 coding agent 的可控性和可监督性。随着 coding agent 能修改大型代码库，单靠自然语言提示很难保证行为边界，约束系统会成为可扩展监督的重要组成。它对应的是实际开发中的权限、规范、测试和代码审查问题。

来源：https://arxiv.org/abs/2607.02389
