# AI Tech Daily

AI 技术热点日报，关注模型、API、开源项目、论文进展、AI agent、图像/视频/多模态生成、算力基础设施、开发者工具和安全评估。

每天按北京时间整理前一天的新闻，文件名使用日期，方便在 GitHub 上直接浏览和归档。

## 编辑规则

- 只收录前一天北京时间 00:00-23:59 发布或更新的内容。
- 跨时区来源按页面标注的本地发布日期判断；不能收录页面日期早于目标日期的旧闻。
- 目标是最多 30 条；不用旧闻补齐数量，如果前一天合格技术新闻不足 30 条，就少发。
- 每条来源必须是具体文章、官方博客、论文页、技术报告、系统卡或 GitHub 项目页。
- 首页、排行榜、搜索页、频道页和聚合页只能用于发现线索，不能作为最终来源。
- 图像生成、视频生成、多模态理解、视觉模型和创作工具链是固定关注方向。
